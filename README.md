# Rock Paper Scissors (MATLAB GUI)

A simple MATLAB GUIDE app to play Rock-Paper-Scissors against a randomized computer opponent. The GUI uses button images and shows the computer choice and result.

## Features
- GUI built with MATLAB GUIDE (.fig)
- Random computer selection
- Result display and reset

## Run
1. Open the folder in MATLAB.
2. Run `Rock_paper_scissors.m` (or open `Rock_paper_scissors.fig` and click Run).
3. Click Rock/Paper/Scissors.

## Notes
Image assets (`rock.png`, `paper.png`, `scissors.png`) must stay next to the `.m` and `.fig` files.
